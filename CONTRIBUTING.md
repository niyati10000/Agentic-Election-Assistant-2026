# Contributing to National Election Safety Agent

We welcome contributions! Please follow these guidelines:

## How to Contribute
1. Fork the repository.
2. Create a new branch for your feature or bugfix.
3. Ensure your code passes all tests in the `tests/` directory.
4. Submit a Pull Request with a clear description of your changes.

## Code of Conduct
Please maintain a respectful and professional environment. Focus on accuracy, security, and accessibility.
